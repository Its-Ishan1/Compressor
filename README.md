# AVIF Mini - Forensic-Aware Image Compressor

**AVIF Mini** is a premium, high-performance image optimization tool designed specifically for security researchers and AI developers. Unlike standard compressors that prioritize raw file size above all else, AVIF Mini uses a **forensic-aware adaptive algorithm** to reduce image weight while meticulously preserving the digital noise, artifacts, and ML features required for accurate AI image detection.

Built for the next generation of image formats, it seamlessly handles HEIC/HEIF (iPhone), PNG, JPEG, and WebP, converting them into ultra-optimized AVIF files without sacrificing the procedural integrity of the pixels.

## Features
- **Forensic-Aware Logic**: Preserves ML features (noise/artifacts) while reducing file size.
- **HEIC/HEIF Support**: Native support for iPhone photos via `pillow-heif`.
- **Automatic Optimization**: No manual tweaking needed; intelligent thresholds based on input format.
- **AVIF Conversion**: Highly efficient next-gen compression.
- **Glassmorphic UI**: Premium dark-mode interface.


## Tech Stack
- **Backend**: Flask
- **Processing**: Pillow + pillow-heif
- **Frontend**: Vanilla HTML/CSS/JS (Glassmorphic Design)
